SETUP
=====

 1. Install the Python 2.7.12 or Anaconda 4.1.1 (64-bit)

 2. Transfer the files named below into Python installed directory:

	a) PingServer.py
	b) PingClient.py
	c) Readme.txt

EXECUTION PROCEDURE
===================

 1. Migrate the 2 Python script files.
 2. Firstly, execute the script PingServer.py using the below syntax.

    "python PingServer.py <server port no> <seed(optional)>"

For example:

    "python PingServer.py 10005 5"

 3. Secondly, Execute the script PingClient.py using below syntax.

    "python PingClient.py <server ip address> <server port no>"

For example:

    "python PingClient.py 127.0.0.1 10005"

 4. Observe the print statements and ping statistics for each script execution.
 5. Exit the python console.
